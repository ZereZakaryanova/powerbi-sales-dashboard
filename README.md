# 📊 Sales Dashboard (Power BI)

## 📌 Project Overview
Разработка интерактивного дашборда для анализа продаж, прибыли, логистики и клиентов интернет-магазина.

## 🛠 Tools
- Power BI
- DAX
- Data Visualization

## 📊 Что сделано
- Создан интерактивный дашборд
- Реализована аналитика по продажам и прибыли
- Добавлен анализ по складам и логистике
- Визуализированы клиенты по регионам
- Проанализирована эффективность сотрудников

## 📈 Основные метрики
- Выручка: 34,19 млн
- Прибыль: 10,26 млн
- Рентабельность: 30%

## 💡 Insights
- Выявлена сезонность продаж (колебания по месяцам)
- Определены наиболее прибыльные категории товаров
- Найдена разная эффективность складов
- Выявлены регионы с наибольшим количеством клиентов
